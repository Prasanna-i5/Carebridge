<P># 🌉 Carebridge – Sharing Life's Essentials Platform

A responsive web platform designed to connect donors, businesses, and non-profit organizations to reduce waste and help people in need by facilitating the donation of essential items.

🔗 **Live Demo:**  
https://prasanna-i5.github.io/Sharing-life-s-essentials-platform/

---

## 📖 Project Overview

Carebridge is a static web application that aims to bridge the gap between donors and communities by providing an easy-to-use platform for sharing essential resources.

The website presents information about the donation process, platform impact, payment support, business partnerships, and volunteer opportunities through a clean and user-friendly interface.

---

- 🏠 Responsive Home Page
- ❤️ Donation Information
